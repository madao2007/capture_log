# capture_log

## How to use
1. Install python module
```shell
pip3 install -r requirements.txt
```
2. Modify the `device_list.csv`
the latest sring is 'enable' password if need.
3. Modify the `command_list.csv`
4. Run the scripts
```shell
python3 cap_log.py
```
5. Find the logs in `show-output` folder
